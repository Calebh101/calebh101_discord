## What's this?

This is a small framework I built on top of [nyxx](https://pub.dev/packages/nyxx). It has support for automatic token management, built-in settings, and other things.

## Usage

In `pubspec.yaml`:

```yaml
dependencies:
  calebh101_discord:
    git:
      url: https://github.com/Calebh101/calebh101_discord.git
      ref: main
```
